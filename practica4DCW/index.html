Caracteristicas de Grid en Tailwind

Gid Template Colums

Define el numero y tamano de columnas co grid-cols-{n} o cols-[pattern]

Gid Column Span Spam

Controla cuantas columnas ocupa un elemento con col-span{n}

Gid Template Rows

Define filas con grid-rows-{n} o grid-rows-[pattern]

Gap

Añade espacios entre elementos con gap-{size}
